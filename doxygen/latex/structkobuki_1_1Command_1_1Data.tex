\subsection{\-Detailed \-Description}
\doxyref{\-Data}{p.}{structkobuki_1_1Command_1_1Data} structure containing data for commands. 

\-It is important to keep this state as it will have to retain knowledge of the last known command in some instances -\/ e.\-g. for gp\-\_\-out commands, quite often the incoming command is only to set the output for a single led while keeping the rest of the current gp\-\_\-out values as is.

\-For generating individual commands we modify the data here, then copy the command class (avoid doing mutexes) and spin it off for sending down to the device.
