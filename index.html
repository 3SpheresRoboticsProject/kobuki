Introduction

Kobuki documentation is located in several places - this domain hosts only the software driver documentation. This should be of interest if you are either

  • Compiling or cross-compiling the c++ driver and building your own implementation on top.
  • Developing the c++ driver itself.
  • Writing a non-c++ driver implementation for kobuki.

If you are one of these users, head straight to the doxygen. Guides and api documentation can be found therein.

Other Documentation Sources

For other sources of documentation:

  • Official Kobuki Home - about, sales, specifications and hardware howtos.
  • Ros Documentation - kobuki in Ros.
  • Turtlebot Documentation - a full reference platform implementation for kobuki.
